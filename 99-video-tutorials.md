## How to use these videos

1. **First pass - Microsoft Learn / study cram** (one sitting). Walks every skill area in the official outline.
2. **Second pass** - focused channel deep-dives for any topic where you didn't feel confident.
3. **Third pass** - Azure Friday + WAF series for the *why* behind each design choice.
4. Then take the **Practice Assessment** (see the sidebar) before scheduling the real exam.
