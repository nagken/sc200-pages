# SC-200 Concept and Reference Index

> Quick-lookup index of every term, service, and acronym you will see on the exam.

- **Defender XDR** - Cross-workload detection + response (M365)
- **Defender for Endpoint** - EDR/AV on devices
- **Defender for Office 365** - Email + Teams threat protection
- **Defender for Identity** - On-prem AD signals + UEBA
- **Defender for Cloud Apps** - CASB + SaaS app posture
- **Defender for Cloud** - CSPM + CWP for Azure/AWS/GCP
- **Sentinel** - SIEM/SOAR
- **Advanced Hunting** - KQL across XDR tables
- **Attack Disruption** - Automated containment during attack
- **AIR** - Automated Investigation + Response
- **Fusion** - Multi-stage attack detection in Sentinel
- **UEBA** - User and entity behavior analytics
- **Watchlist** - Reference table for hunts/rules
