# Glossary and Acronyms - SC-200

| Term | Definition |
|---|---|
| **XDR** | Extended Detection and Response |
| **SIEM** | Security Information and Event Management |
| **SOAR** | Security Orchestration, Automation and Response |
| **AIR** | Automated Investigation and Response |
| **UEBA** | User and Entity Behavior Analytics |
| **KQL** | Kusto Query Language |
| **IOC / IOA** | Indicator of Compromise / Indicator of Attack |
| **Fusion** | Sentinel multi-stage ML correlation |
| **Watchlist** | Reference table for hunting/rules |
| **Bookmark** | Saved hunting result with context |
| **Livestream** | Real-time hunting query |
| **Attack Disruption** | Automated containment during in-progress attack |
